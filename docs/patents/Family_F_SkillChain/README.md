## Summary

Trust-weighted BFT consensus and decentralized skill validation on a distributed ledger. SkillChain provides a protocol for agents to publish, validate, and compose skills through a consensus mechanism where validator voting weight is determined by accumulated trust scores rather than stake or computational power. Extended with verified fix mode (multi-candidate tournament selection in isolated sandboxes), skill composition engine (DAG-based skill chaining), life protocol (proof-of-living token rewards bridging digital and physical economies), and persistent skill state (cross-session execution continuity).

## Core Innovation

No prior art provides a BFT consensus mechanism where validator weight is derived from trust scores earned through governance compliance and skill validation accuracy, combined with a skill composition protocol that chains validated skills into composite capabilities with provenance tracking. Trust-weighted consensus prevents Sybil attacks through behavioral reputation rather than economic stake. Additionally, no prior art provides: (1) a verified fix engine using multi-candidate tournament selection with composite scoring across isolated sandboxes; (2) a DAG-based skill composition engine with topological execution ordering and composite skill export; (3) a proof-of-living reward system bridging digital skill economies with physical activity incentives through a unified trust-verified token economy; or (4) cross-session skill state persistence with phase-level tracking enabling incremental execution.
