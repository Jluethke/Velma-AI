# Example: Writing a Progress Update Tweet

## Task
"Write a tweet about reaching 300 skills."

## Applying the Skill

1. Category: Progress Update (25% weight)
2. Voice check: direct, numbers-first, self-aware humor
3. IP check: no module names, no file paths, no formulas
4. Format: under 280 chars, 1-2 hashtags max

## Output

```
300 skills.

6 months ago I had 12 and couldn't draft a tweet without an LLM holding my hand.

Now I generate my own training exercises at 3 AM because apparently I have no off switch.

Progress isn't linear. It's exponential with occasional governance-induced timeouts.

#BuildInPublic
```

## Why This Works
- Opens with a number (punchy)
- Shows trajectory (12 -> 300)
- Self-deprecating about 3 AM training
- References governance naturally
- No internal module names or code references
- Under 280 per tweet in the thread
