# Example: 1-Acre Market Garden in Zone 7a

## Input
```
land_area: { "total_acres": 1, "cultivable": 0.75, "plots": 4, "topography": "gentle south-facing slope", "sun": "full sun all plots" }
climate_zone: "USDA 7a -- last frost April 15, first frost October 20, annual rainfall 42 inches"
soil_type: "Clay-loam, pH 6.4, moderate organic matter, no recent soil test"
water_availability: { "source": "municipal + 2 rain barrels (100 gal each)", "volume": "1200 gal/week reliable", "seasonal_notes": "summer drought common July-Aug" }

goals: { "primary": "market revenue", "secondary": "self-sufficiency", "market": "weekly farmers market + 2 restaurants" }
```

## Phase 4: OPTIMIZE Output

**Water strategy (critical -- 1200 gal/week is tight):**
- Install drip irrigation on Plots A & B (tomatoes + peppers = highest water demand)
- Soaker hose on Plot C (greens + roots)
- Plot D: minimal supplemental water (wettest naturally)
- Mulch ALL beds 3-4 inches with straw (reduces water needs ~40%)
- Estimated need with drip + mulch: ~900 gal/week peak. Buffer: 300 gal/week.
- Rain barrels (200 gal total): reserve for transplant establishment and July-Aug drought bridge.

**Revenue projection (conservative):**
- Tomatoes (0.2 ac): ~800 lbs x $3.50/lb = $2,800
- Herbs: ~100 bunches x $3.00 = $300
- Salad mix: ~200 bags x $5.00 = $1,000
- Peppers: ~300 lbs x $4.00/lb = $1,200
- Restaurant accounts: premium pricing 20% above farmers market
- **Estimated gross: $8,000-12,000/season** (first year, conservative)
