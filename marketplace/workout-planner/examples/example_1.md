# Example: Intermediate Hypertrophy Program (4 days/week, Full Gym)

## Input
```
fitness_goals: ["hypertrophy", "strength"]
available_equipment: ["full gym (barbell, dumbbells, cables, machines)"]
schedule: { "days": 4, "duration": "75 min", "days_available": ["Mon", "Tue", "Thu", "Fri"] }
current_fitness_level: "intermediate"
injuries_limitations: ["mild lower back sensitivity -- avoid heavy conventional deadlifts"]
```

## OBSERVE Output
**Methodology:** Hypertrophy-primary with strength foundation. 8-12 rep range emphasis, compound-first.
**Split:** Upper/Lower x2 (Mon Upper, Tue Lower, Thu Upper, Fri Lower). 48h+ rest between same muscle groups.
**Equipment:** Full pool available.
**Contraindications:** No conventional deadlift. Substitute: trap bar deadlift, Romanian deadlift (lower load on lumbar spine).

## REASON Output
**Volume targets:** 14-16 sets per major muscle group per week (intermediate).
- Chest: 16 sets (priority)
- Back: 16 sets (priority)
- Quads: 14 sets
- Hamstrings: 12 sets
- Shoulders: 12 sets (some overlap with pressing)
- Arms: 10 sets (overlap from compounds)

**Progression:** Double progression. Rep range 8-12. When you hit 12 reps on all sets, increase weight 2.5-5 lbs and restart at 8 reps.

**Mesocycle:** 4-week accumulation + 1-week deload. Volume increases +1 set per muscle group per week across the mesocycle.

### Deload Week (Week 5)
Same exercises, same weights, 50% volume (cut all sets in half, round up). RPE 6-7 max.

### Progression Rule
"If you hit the top of the rep range on all working sets (e.g., 4x12 where range is 8-12), add 5 lbs to barbell exercises or 2.5 lbs to dumbbell exercises next session. Reset to bottom of rep range (8 reps) and build back up."
