# SkillChain

Decentralized AI Skill-Sharing Network. Publish, discover, validate, and trade AI skills across any agent platform.

## Install

```bash
pip install skillchain
```

## Quick Start

```bash
# Initialize and connect to Claude Code
skillchain mcp install

# Or run the MCP server directly
skillchain mcp serve
```
